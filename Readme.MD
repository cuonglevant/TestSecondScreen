Phần mềm này được phát triển có sử dụng Virtual Display Driver: https://github.com/VirtualDrivers/Virtual-Display-Driver

Các bước cài đặt và sử dụng:

! Lưu ý, vui lòng bật chế độ gỡ lỗi bên trong cài đặt nhà phát triển trên điện thoại.

1. Chạy setup_pc.bat để cài đặt python và VDD.
2. Sau khi chạy, nhấn install driver. ![Minh họa](image.png)
3. Vào Windows Setting, chọn Display, chỉnh cài đặt của màn hình 1 thành 1920\*1080 (Lưu ý, chỉ hỗ trợ 1 màn hình chính + 1 màn hình phụ)
4. Kết nối với điện thoại bằng cáp USB, bật chế độ gỡ lỗi và cho phép cài đặt từ USB trong chế độ nhà phát triển; sau đó chạy setup_phone.bat để cài đặt USB driver và app cho Android.
5. Chạy start.bat
