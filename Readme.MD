Các bước cài đặt và sử dụng:

1. Giải nén file VDD Control và chạy VDD Control.exe
2. Sau khi chạy, nhấn install driver.
   ![Minh họa](image.png)
3. Vào Windows Setting, chọn Display, chỉnh cài đặt của màn hình 1 thành 1920\*1080 (Lưu ý, chỉ hỗ trợ 1 màn hình chính + 1 màn hình phụ)
4. Cài đặt Python 3.13.2
5. Chạy file start.bat
   Nếu xuất hiện lỗi, kiểm tra cài đặt ADB driver
